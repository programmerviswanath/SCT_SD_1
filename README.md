Task-1
Create a program that converts temperatures between Celsius, Fahrenheit, and Kelvin using a user-friendly GUI.

Description
This project is a Tkinter-based temperature converter that allows users to:
Enter a temperature value
Choose the input scale (Celsius / Fahrenheit / Kelvin)
Convert it instantly to the other two scales
The application has a clean interface and uses standard conversion formulas.

Technologies Used
1) Python
2) Tkinter (GUI)

Features
Simple and modern GUI
Dropdown selection for temperature unit
Automatic conversion
Error handling for invalid inputs
